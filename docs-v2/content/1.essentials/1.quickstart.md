### Installation

::code-group
```bash [npm]
npx nuxi module add konkonam/nuxt-shopify
```

```bash [pnpm]
pnpm dlx nuxi module add konkonam/nuxt-shopify
```

```bash [bun]
bun x nuxi module add konkonam/nuxt-shopify
```

::

### Configuration
```ts [nuxt.config.ts]
export default defineNuxtConfig({
    shopify: {
        name: 'quickstart-abcd1234',
            clients: {
                storefront: {
                apiVersion: '2025-04',
                publicAccessToken: 'YOUR_ACCESS_TOKEN',
            },
        },
    },
})
```

::alert{type="info"}
Restart your dev server after changing the configuration, it will automatically fetch the introspection schema.
::

### Usage

#### Fetch products in a nitro endpoint
```ts [~/server/api/products.ts]
export default defineEventHandler(async (event) => {
    const storefront = useStorefront()
    const { data } = await storefront.request(`#graphql
        query Products {
            products(first: 10) {
                edges {
                    node {
                        id
                        title
                    }
                }
            }
        }
    `)

    return data
})
```

##### Display products in a page:
```vue [~/app/pages/index.vue]
<script setup lang="ts">
const { data } = await useFetch('/api/products')
</script>

<template>
    <pre>{{ data }}</pre>
</template>
```
